Prospektverteiler-Software ZSP

Die Logistik-Software ZSP wird seit über 15 Jahren erfolgreich von Prospektverteiler-Unternehmen eingesetzt.

Technische Basis

Das Programm hat eine 2-Tier-Architektur, bestehend aus Frontend-Programm und Datenbank. Das Frontend ist ein Windows-Client (geschrieben in SAP Sybase PowerBuilder), die Datenbank ist eine SAP SQL Anywhere-Datenbank.
